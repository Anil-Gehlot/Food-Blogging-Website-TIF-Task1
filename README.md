# Pixel Perfect Responsive UI Build

This project is a challenge to create pixel-perfect responsive webpages using **React.js/Next.js** based on a Figma design. The UI should adapt seamlessly between mobile and desktop screens, providing an optimal viewing experience across different device sizes.

## Requirements

- **Framework**: React.js/Next.js
- **Styling**: Preferred solution is CSS-in-JS with Chakra UI.
- **Responsiveness**:
  - **Mobile Layout**: 375px width
  - **Desktop Layout**: 1440px width
- **Design**: All colors, fonts, and design elements must match those specified in the provided Figma file.

## Features

1. **Pixel Perfect Design**:
   - The layout and styling must precisely match the Figma design.
   - Elements should be identical in spacing, alignment, and visuals.
